## Repository of many skeletons to build Docker images

Just change to the desired directory and run:

```
docker build -t <image>:<tag> .
```

Where image:tag must be of your own choice.

Don't forget the dot (.) at the end.

Note: almost all subdir here have a docker hub automated build. Check in
[https://hub.docker.com/?namespace=lrgc01](https://hub.docker.com/?namespace=lrgc01)
